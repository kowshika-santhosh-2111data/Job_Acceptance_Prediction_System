#JOB ACCEPTANCE PREDICTION SYSTEM
This project predict whther a candidate will be placed or not using machine learning.
It also provides data analysis and insights to help recruiters make better hiring decisions.

---
TOOLS 
- python
- pandas, NumPy
- Matplotlib, Seaborn
- Scikit - learn
- MySql
- Stearmlit

---
#WORKFLOW
1. Data Collection
2. Data Celaning & Preprocessing
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Buliding
6. Model Evaluation
7. Dashboard Development

---
KEY INSIGHTS
- Skills match, interview performance, and experience are major factors influencing job acceptance  
- Higher skills and interview scores increase selection chances  
- Employment gaps and high salary expectations reduce acceptance rates  
- The model effectively predicts job acceptance and supports better hiring decisions
